# Nouri shop

در این دوره، برنامه‌نویسی اندروید را به‌صورت کاملاً پروژه‌محور و در قالب ساخت یک اپلیکیشن فروشگاه آنلاین مشابه دیجی‌کالا یاد گرفتیم. این پروژه از صفر تا صد، با استفاده از معماری MVVM و جدیدترین تکنولوژی‌های اندروید پیاده‌سازی شده است تا مهارت‌های واقعی توسعه 
اپلیکیشن را در سطح حرفه‌ای کسب تمرین کنیم .

.

⚙️ تکنولوژی‌ها و ابزارهای مورد استفاده

* Kotlin – زبان برنامه‌نویسی اصلی

* MVVM Architecture – معماری مدرن و قابل نگهداری

* ViewModel, LiveData, Flow – مدیریت داده و چرخه حیات

* Coroutines – مدیریت تردها و عملیات هم‌زمان

* Dagger-Hilt – تزریق وابستگی (Dependency Injection)

* DataStore – ذخیره‌سازی داده‌های کاربر

* Retrofit + OkHttp + Interceptor – ارتباط با API و مدیریت درخواست‌ها

* Material Design + Motion Layout – طراحی حرفه‌ای و انیمیشن‌های پویا

* Navigation Component – مدیریت ناوبری بین صفحات

* Coil / Glide – بارگذاری تصاویر

* Gson – مدیریت JSON

* Lottie / Shimmer – انیمیشن و Skeleton Loading

* MPAndroidChart – نمایش نمودار

* Persian Date Picker – تقویم فارسی.
